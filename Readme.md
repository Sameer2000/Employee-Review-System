# TECHSTACK
- Node.js
- Express
- Passport
- EJS
- MongoDB

# SETUP

Run `npm install` to install required dependencies

Create .env with following variables
- PORT=`Your Port`
- SECRET=`Your Secret`
- MongoURL=mongodb://localhost:27017/`Your DB Name`

Run `nodemon index.js`
